# Security Policy

## Supported Versions

This project supports the latest published release and recent prereleases from the `main` branch.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Current prereleases | Yes |
| Older prereleases | No |

## Reporting a Vulnerability

Use GitHub Private Vulnerability Reporting for this repository:

https://github.com/lammesen/codex-brave-web-search/security/advisories/new

Do not open public issues for security reports.

When possible, include:

- affected version(s)
- reproduction steps
- impact assessment
- suggested remediation
